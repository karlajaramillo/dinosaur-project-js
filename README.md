# Object Oriented Javascript 

## Dinosaur infographic

### Project Introduction

Let's create an infographic based on your information and compare yourself with dinosaurs! After you fill in the form, you will discover different species of dinosaurs and random key facts. 

### Aim of the Project

Build an infographic with 9 tiles where human information is located in the center. This infographic will compare human data to dinosaur data, precisely: height, weight and diet. The dino tiles contain the species, an image and a fact that is chosen at random. This project is based on Object Oriented Javascript.
